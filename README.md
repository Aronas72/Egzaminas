# Pasikartojančių žodžių ir nuorodų iširnkimas iš duomenų failų

Ką ši programa daro:
1. Nuskaito .txt failus ir išrenka žodžius, kurie pasikartoja daugiau nei 1 kartą
2. Iš kito .txt failo išrenka visas nuorodas su visomis galimomis galūnėmis ir išveda į atskirą rezultatų failą
